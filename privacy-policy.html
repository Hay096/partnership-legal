Política de Privacidade — PartnerShip

Última atualização: 13 de julho de 2026

Esta Política de Privacidade descreve quais dados o bot PartnerShip ("o Bot", "nós") coleta ao ser usado em um servidor Discord, como esses dados são usados, armazenados e por quanto tempo são retidos. Ao adicionar o Bot ao seu servidor ou interagir com seus comandos, você concorda com as práticas descritas aqui.

1. Dados que coletamos

Ao ser configurado em um servidor, o Bot armazena:

  • ID do servidor (guild ID), ID do canal e ID da categoria configurados para parcerias.
  • O texto promocional configurado pelo administrador do servidor.
  • Tags de interesse selecionadas pelo administrador, usadas para pareamento com outros servidores.
  • IDs de mensagens de parceria publicadas pelo Bot, usados internamente para permitir renovação e recuperação em caso de exclusão acidental do canal.
  • Links de convite extraídos de mensagens explicitamente designadas pelo administrador durante o comando de migração manual (/migrar-parceria).
  • Nomes de canais, categorias e cargos públicos, e a descrição do servidor, usados apenas na verificação automatizada de conformidade com as Community Guidelines do Discord.

Sobre o Message Content Intent: quando ativo, o Bot processa o conteúdo de mensagens apenas nestes dois casos específicos:

  • Mensagens no canal designado como origem durante /migrar-parceria, exclusivamente para extrair links de convite já postados ali pelo próprio administrador.
  • A mensagem de uma parceria manual, exclusivamente para identificar uma possível menção de representante do servidor.

O Bot não armazena o texto completo dessas mensagens. Ele extrai apenas o dado relevante (o link de convite, ou o ID do usuário mencionado) e descarta o restante do conteúdo imediatamente após o processamento.

2. Dados que não coletamos

  • O Bot não lê, processa ou armazena o conteúdo de mensagens de usuários fora dos dois fluxos explícitos descritos acima.
  • O Bot não coleta dados de presença (status online, offline, ausente, ou atividade em jogos).
  • O Bot não coleta, armazena ou processa conteúdo de canais privados ou invisíveis para @everyone.
  • O Bot não vende, aluga, compartilha ou transfere dados a redes de anúncio, data brokers, ou qualquer serviço de terceiros não essencial à sua operação.

3. Como usamos os dados

Os dados coletados são usados exclusivamente para operar as funcionalidades do Bot: parear servidores compatíveis com base em tags de interesse, publicar e renovar mensagens de parceria mutuamente configuradas, recuperar a configuração de um servidor em caso de exclusão acidental do canal de parcerias, e verificar conformidade básica com as diretrizes de conteúdo do Discord para proteger a rede de parcerias contra uso indevido.

4. Retenção de dados

Dados de configuração de um servidor (canal, texto, tags, IDs de mensagem) são mantidos enquanto o Bot permanecer instalado nesse servidor. Ao remover o Bot de um servidor, esses dados são mantidos por até 30 dias, para permitir que o administrador reinstale o Bot sem perder a configuração; após esse período, os dados são excluídos permanentemente. Dados extraídos temporariamente durante o processamento de uma mensagem (como o texto usado para localizar um link de convite) não são retidos após a conclusão da ação correspondente.

5. Compartilhamento de dados

Não compartilhamos dados com terceiros, exceto quando estritamente necessário para operar a infraestrutura do Bot (por exemplo, o provedor de hospedagem do banco de dados), e sempre sob obrigação de confidencialidade equivalente à descrita nesta política. Não usamos os dados coletados para treinar modelos de machine learning ou inteligência artificial.

6. Seus direitos

Administradores de servidor podem, a qualquer momento, remover o Bot do servidor, o que interrompe imediatamente qualquer atividade de parceria envolvendo aquele servidor. Para solicitar a exclusão antecipada dos dados de um servidor, ou esclarecer dúvidas sobre esta política, entre em contato pelos canais listados na seção 8.

7. Alterações a esta política

Esta política pode ser atualizada periodicamente. Alterações relevantes serão comunicadas através do servidor de suporte oficial do Bot. O uso continuado do Bot após uma atualização constitui aceitação da política revisada.

8. Contato

Para dúvidas sobre esta Política de Privacidade ou sobre o tratamento de dados pelo PartnerShip, entre em contato através do nosso servidor de suporte oficial no Discord.
